# 📘 Student Grades Tracker – Excel Project

A simple but effective Excel project designed to analyze student performance, calculate averages, assign letter grades, and visualize class trends.  
This project was created after completing the **Introduction to Excel** course on DataCamp, applying the core skills learned in a practical, real‑world scenario.

---

## 🎯 Project Overview

The **Student Grades Tracker** allows you to:

- Record student scores across multiple assignments  
- Automatically calculate averages and final grades  
- Assign letter grades using formulas  
- Highlight performance patterns using conditional formatting  
- Visualize grade distribution with charts  
- Sort and filter students based on performance  

This project demonstrates foundational Excel skills essential for data analysis.

---

## 🧠 Skills Used & Learned

### 🔹 1. Data Cleaning & Organization
- Structured tables for clean data entry  
- Proper formatting of headers and data types  
- Sorting and filtering student records
- Use of different sheets for data on different Subjects
- Removing duplicates  

### 🔹 2. Essential Excel Formulas
- `AVERAGE()` for computing student averages  
- `COUNTIF()` for grade frequency analysis  
- `IF()` statements for assigning letter grades  
- `MIN()`, `MAX()`, and `SUM()` for summary statistics  

### 🔹 3. Conditional Formatting
- Color‑coding high and low performers  
- Highlighting failing grades  
- Visual cues for quick interpretation  

### 🔹 4. Data Visualization
- Bar chart showing grade distribution  
- Line or column chart for assignment performance  
- Clean, readable visuals for reporting  

### 🔹 5. Basic Dashboard Layout  
- Beginner‑friendly dashboard design

## 🚀 How It Works

1. Enter student names and assignment scores into the table.  
2. Excel automatically calculates:  
   - Average score  
   - Final grade  
   - Letter grade  
3. Conditional formatting highlights performance levels.  
4. Charts update dynamically as new data is added.
